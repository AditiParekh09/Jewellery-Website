# Unai Jewellers — Website

A responsive, luxury-themed static website for Unai Jewellers, a Gold & Silver showroom based in Unai, Gujarat, India.

## Features
- 🏠 Home page with hero banner and live gold/silver rate ticker
- 💍 Searchable & filterable jewellery collection catalogue
- 📖 About/heritage page with animated timeline & counters
- ❓ FAQ accordion
- 📞 Contact page with form validation & embedded map
- 📱 WhatsApp click-to-chat integration
- 📱 Fully responsive (mobile, tablet, desktop)

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Bootstrap 5.3
- Font Awesome 6
- AOS (Animate on Scroll)
- Google Fonts (Playfair Display, Poppins)
